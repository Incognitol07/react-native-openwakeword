# Example App

A testing environment for the wake word detection engine.

### Usage
- **Benchmark**: Observe the "Room Noise" bar to check microphone levels.
- **Speak**: Say "Alexa" and watch the "Voice Clarity" peak.
- **Tune**: Use the threshold buttons to find the right sensitivity for your environment.
- **Deploy**: Use your calibrated threshold value in your main application.

### Development
```bash
npm install
npm run android # or npm run ios
```
